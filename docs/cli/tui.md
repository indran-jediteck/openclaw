# `lynkai tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
lynkai tui
lynkai tui --url ws://127.0.0.1:18789 --token <token>
lynkai tui --session main --deliver
```
